The Recipe Section displays all the recipes used for ThirtyMeals. You may notice that there are only 28 recipes. This is because ThirtyMeals assumes a 60,000 calorie total for a 30-day month. If each meal was exactly 2,000 calories, then there would indeed be 30 meals. As it worked out, some of the meals have more than 2,000 calories, so it only took 28 meals to hit the 60,000 calorie total. However, ThirtyMeals sounds better than TwentyEightMeals, so the name will remain.
