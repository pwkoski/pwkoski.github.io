ThirtyMeals is a program that calculates how much growing area and storage a person would need to grow their own food for a year. The program takes in a list of meals, a list of ingredients, and a monthly individual caloric goal (60,000 cals). Assuming you eat nothing but the meals in the list, ThirtyMeals will calculate information related to:

  • Growth Area
  • Storage
  • Nutrition
  • Growth Schedule (NEW)

It uses the Python programming language for almost all calculations. Running Python in the browser is slow, so each page will take some time to load. Each page will have something (chart, graph) to look at other than just text, so if you don't see anything...just wait!

The initial calculations are complete when you see a table of the ingredient amounts below this paragraph. After you see the table, feel free to click on any of the other pages.
